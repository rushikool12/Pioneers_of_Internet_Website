Internet
A global computer network infrastructure linking smaller computer networks used for trasnmission/reception of different types of data/media using standardized protocols
World Wide Web
The World Wide Web is an information system that links the internet to the computer using hypertext and allows user to access images and documents globaly shared over the internet.
Hypertext
A term coined by Ted Nelson in 1960s refers to a text that can be linked to other texts and it is not constraint to be text it can include graphics,videos and sound too.
Multimedia
It combines several forms of media such as text,graphics,drawings, video, animation and any other media that can be represented,stored,tramsmitted and processed digitally.
CLI
Command Line Interface is a text-based interface used for operation of software and OS while allowing user to respond to visual prompts by typing single commands on keyboard.
GUI
Graphical User Interface is an interface which allows user to interact with Computers, Mobile Phones and Other Devices by using icons, menus and other visual indicators and user will use pointing devices such as mouse, touch screen to control the software.
HCI
Human Computer Interaction is the field of study of how people interact with computer and to find ways that will improve this interaction.
FTP
File Traansfer Protocol is a standard internet protocol for transmitting files between computers on the internet over TCP/IP connections.
SSH
Secure Socket Shell is a network protocol which allows to user to access a remote computer. SSh provides a strong authenticatino and secure encrypt data communications between two computers connecting the over an insecure network such as Internet.
HTTP
HyperText Transfer Protocol is an appplication protocol runs on TCP/IP and it is used for transmission of files (text, graphic images,sound, video and other multimedia files) on World Wide Web.
Telnet
It is a user command underlying TCP/IP protocol for acccessing remote computers that are part of the same network.
HTML
Hypertext Markup Language is a text-based language used to define how a Webpage and its contents such as text, images will be displayed on the browser.
CSS
Cascading Style Sheets are used to format the layout of the Webpages. They are used to define styles, colors , fonts, sizes and other aspects of a Webpage.
