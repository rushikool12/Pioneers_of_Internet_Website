Origins of the Web

The Internet, widely used platform by billions of people nowadays to share knowledge, information, art, music and lots of content. The website you are currently watching is a part of the internet. Have you ever wondered how it all began?

The internet which is referred as "networks of networks" was emerged in 1960s in USA and can be accessed by anyone who connects to it. Its used for emails, chats, sending/recieving of audio/video files etc. It has given birth to varoius fields such as e-commerce and e-bussiness.

The need of the computer networks began in 1960 when people started working on the computers. As each one had differnet data in it there was a need of data sharing in real time i.e Time-Sharing. Time-Sharing system allowed a computer's resources to be shared in rapid succession with multiple users, cycling through the queue of users so quickly that a computer appeared dedicated to each users tasks despite the existence of many others accessing the system "simultaenously". ARPANET ( a US Defence Department Project) established first host to host connection on October 29, 1969. ARPANET used packet switching to allow multiplecomputers to communicate on a single network. In 1970s scientist Robert Kahn and Vincet Cerf developed Transmission Control Protocol and Internet Protocol, or TCP/IP. ARPANET adopted TCP/IP on January 1, 1983, and from there researchers began to assemble the “network of networks” that became the modern Internet.

Later, Tim Berners-Lee a Software Engineer at CERN invented the World Wide Web in 1990s. In 1989,CERN was the largest internet node in Europe, and Berners-Lee saw an opportunity to join the Hypertext with the Internet. The World Wide Web enabled the pathways to transfer text, images, graphics, videos and many more stuff. Berners-Lee designed a GUI browser which allowed the user to point to a link that would carry the user to next information. Nicola Pellow one of the associates of Berners-Lee designed the first line-mode web browser which allowed connnection to Telnet but did not support the mouse click.

Many people think that Internet and World Wide Web are the same thing but they are not. Internet basically allows computers around the world communicate with each other. It is the hardware part, whereas the WWW is the software part which uses protocols (HTTP) and Hypertext to share, see and navigate through the pages available.
